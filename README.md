# Pulse

Pulse is a personalized sports and culture intelligence platform that helps users track the teams, artists, and topics they care about, then delivers live context, recent developments, and discovery in one place.

## Why I Built This
As a learning activity and dive deeper into some technologies I want to study. I wanted to build a system that combines my interests in sports, music, and pop culture into one intelligent product experience. Pulse aims to create a single personalized hub.

## Core Features
- Track favorite teams, players, artists, and topics
- View recent and upcoming sports updates
- Summarize recent developments around followed teams
- Surface discovery cards across sports, music, and culture
- Build toward lightweight interactive experiences like quizzes and games

## Tech Stack
- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: FastAPI, Python
- Database: PostgreSQL
- Cloud: AWS
- AI: Summarization and retrieval-based context features

## MVP Scope
- Personalized interest tracking
- Sports updates for followed teams
- Recent discussion/news summaries
- Discovery feed for sports and culture

## Stretch Goals
- Guess the Footballer game
- Daily interest briefing
- Retrieval-powered contextual Q&A
- Recommendation improvements
