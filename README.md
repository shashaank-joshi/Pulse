# Pulse

Pulse is a personal sports and music intelligence platform that helps me track favorite teams, matches, artists, songs, and saved trivia/notes in one place.

## Why I Built This
As a learning activity and dive deeper into some technologies I want to study. I wanted to build a single system that combines structured data, personal knowledge capture, search, and AI summaries, while also bringing my interests in play.

## Core Features
- Save favorites
- Track football updates
- Save music and trivia notes
- Search across stored content
- Generate AI summaries

## Tech Stack
- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: FastAPI, Python
- Database: PostgreSQL
- Cloud: AWS
- AI: LLM-based summary generation

## MVP Scope
- FC Barcelona-focused football tracking
- Manual music/trivia saves
- Search across notes and updates
- One AI summary feature
