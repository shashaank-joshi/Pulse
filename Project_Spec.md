# Pulse

## Problem
I follow sports, music, and like knowing about random interesting information across many different platforms, but it is difficult to keep track of what matters, revisit older insights, and search across everything in one place.

## Solution
Pulse is a personal intelligence platform that helps me track my favorite teams, matches, players, artists, songs, and saved notes/trivia in one place. It combines structured data, personal notes, search, and AI-generated summaries so I can quickly understand updates and revisit information I care about.

## Target User
The Target user is just me for now. Maybe, in the future, it could be useful for sports and music fans who want a personal dashboard for tracking interests and saving knowledge.

## MVP
- Add and manage favorites
- Fetch and store football data
- Store music-related items and personal notes
- Save trivia / random TIL notes
- Search across saved content
- Show a simple dashboard
- Generate one AI summary from saved or fetched data

## Success Criteria
- The app is deployed and usable
- I can save favorites and notes
- I can view sports updates in one place
- I can search across my stored data
- I can generate a useful summary
- I can clearly explain the architecture, tradeoffs, and future roadmap

## Non-Goals
- No multi-user support
- No social features
- No mobile app
- No multi-agent architecture in V1
- No support for too many sports in V1
